# 📝 Next Word Prediction using Deep Learning

A Deep Learning-based Natural Language Processing (NLP) application that predicts the next word in a sentence using Recurrent Neural Networks (RNN) and Long Short-Term Memory (LSTM) architectures. The model learns contextual relationships between words to generate accurate next-word predictions.

---

## 🚀 Features

- Predicts the next word based on an input sequence.
- Text preprocessing and normalization.
- Tokenization and vocabulary generation.
- Sequence generation and padding.
- Word embeddings using Keras Embedding layer.
- Comparison of Simple RNN and LSTM models.
- Interactive Streamlit interface for real-time predictions.
